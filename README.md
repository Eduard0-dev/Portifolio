# 📌 Portfólio Pessoal

## 📖 Descrição

Este projeto é um **site de portfólio pessoal** desenvolvido em HTML, CSS e JavaScript, com o objetivo de apresentar informações profissionais, habilidades, projetos e formas de contato.

O site conta com um design moderno, responsivo e organizado, ideal para desenvolvedores que estão iniciando na área e desejam mostrar sua evolução e trabalhos.

---

## ⚙️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Google Fonts (Poppins)

## 🌐 Seções do Site

O portfólio é dividido nas seguintes seções:

* **Header (Menu de navegação)**
  Navegação entre as seções com menu responsivo.

* **Hero (Início)**
  Apresentação principal com nome, descrição, botões e redes sociais.

* **Sobre**
  Informações pessoais, objetivos e resumo profissional.

* **Habilidades**
  Tecnologias e competências utilizadas no desenvolvimento.

* **Projetos**
  Exibição de projetos com descrição, tecnologias e links.

* **Contato**
  Informações de contato e formulário para envio de mensagens.

* **Footer**
  Direitos autorais e botão de voltar ao topo.

```
index.html
```

3. Execute em qualquer navegador

---

## ⚠️ Observações Importantes

* A imagem de perfil (`foto eu.jpg`) deve estar no caminho correto:

  ```
  /foto eu.jpg
  ```

* Os links dos projetos estão como `#`:

  * Substitua pelos links reais (deploy ou GitHub)

* O formulário de contato não possui backend:

  * É necessário integrar com um serviço (ex: EmailJS, Firebase, etc.)

* O menu mobile depende do `script.js`:

  * Certifique-se de que o arquivo está corretamente linkado

## 🧠 Funcionalidades

* Layout responsivo (mobile, tablet e desktop)
* Menu hambúrguer para dispositivos móveis
* Efeitos de animação (reveal)
* Efeito de digitação no título
* Scroll suave entre seções
* Formulário de contato com validação básica
* Botão “voltar ao topo”
